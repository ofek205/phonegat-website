# PHONE GAT — תוכנית SEO (רמה גבוהה, מובנית בכל שלב)

> יעד-על: כל חיפוש של קנייה/תיקון/תקלה סלולר באזור קרית גת → פון גת ראשון. SEO לא מקטע — הוא נבנה לתוך כל שלב.

## 1. מילות מפתח (עברית, כוונה מקומית)
**ליבה (Primary — כוונה מסחרית מקומית):**
- תיקון סלולר קרית גת · מעבדת סלולר קרית גת · חנות סלולר קרית גת
- תיקון אייפון קרית גת · תיקון סמסונג קרית גת
- החלפת מסך אייפון/סמסונג קרית גת · החלפת סוללה קרית גת
- קניית מכשיר/טלפון קרית גת · סלקום קרית גת / נקודת שירות סלקום קרית גת · ניוד קו קרית גת

**משני:**
- תיקון שקע טעינה · תיקון נזקי מים · אביזרים לסלולר קרית גת · העברת נתונים · מכשירים יד 2

**Long-tail (מזין את ה-FAQ + AI Overviews):**
- כמה עולה החלפת מסך אייפון · כמה זמן לוקח תיקון מסך · יש אחריות על תיקון סלולר · איפה מתקנים אייפון בקרית גת · אפשר לקנות מכשיר בתשלומים

## 2. On-Page (לכל מקטע יש תפקיד SEO)
- **Title (≤60 תווים):** `פון גת | תיקון וקניית סלולר בקרית גת — מעבדה ותיקה`
- **Meta description (≤155):** `מעבדת הסלולר הוותיקה בקרית גת (30+ שנה). תיקון מסך וסוללה לכל המכשירים, קניית מכשירים בתשלומים, סלקום וכל החברות. חייגו 052-589-3366.`
- **H1 יחיד:** `פון גת — כל עולם הסלולר בקרית גת` (עם מילת מפתח מקומית).
- **היררכיית H2/H3:** כל מקטע = H2 עם מילת מפתח (שירותים / תיקון / קניית מכשיר / סלקום / אבזור / שאלות נפוצות).
- טקסט אמיתי ועשיר (הסיפור + הביקורות **כטקסט**, לא רק צילומים) = תוכן ייחודי ל-SEO.

## 3. Local SEO (הכי חשוב לעסק מקומי)
- **Google Business Profile:** לתבוע את הפרופיל הקיים (537 ביקורות!) — לא ליצור חדש. NAP עקבי לחלוטין.
- **NAP אחיד** בכל מקום (אתר, GBP, easy, דפי זהב, B144): *פון גת · רחבת תשרי 2, קרית גת 8202175 · 052-589-3366 / 08-681-2050*.
- **Schema (JSON-LD):** `LocalBusiness` / `MobilePhoneStore` — שם, כתובת, טלפון, geo, `openingHours`, `aggregateRating` (4.9), `priceRange`, `sameAs` (סושיאל).
- **מפה מוטמעת** + כפתורי וייז/גוגל.
- דירוגים כטקסט (4.9 / 537) — לא רק צילום.

## 4. Technical SEO
- **Astro SSG** → HTML סטטי, **Core Web Vitals ירוקים** (LCP≤2.5s, INP≤200ms, CLS≤0.1) — סיגנל דירוג.
- Mobile-first (רוב החיפושים בנייד).
- `sitemap.xml` + `robots.txt` + canonical + `lang="he"` + `dir="rtl"`.
- תמונות: `alt` תיאורי (מילות מפתח טבעיות) + lazy-load + srcset + פורמט מודרני (WebP/AVIF).
- HTTPS/HSTS.

## 5. Structured Data
- `LocalBusiness` (חובה — הכי בעל-ערך).
- `FAQPage` על מקטע ה-FAQ. ⚠️ **הערה מציאותית:** מאז 2023 גוגל צמצם תוצאות עשירות ל-FAQ (בעיקר לאתרי ממשל/בריאות). לכן הערך של ה-FAQ הוא **כיסוי מילות מפתח + AI Overviews + רלוונטיות**, לא כוכבים בתוצאות. עדיין שווה להטמיע.
- `BreadcrumbList` (אם רלוונטי), `Organization`.

## 6. GEO / AI Overviews (מנועי תשובות — 2025-2026)
- ישות ברורה: מי/מה/איפה בטקסט מפורש (פון גת, מעבדת סלולר, קרית גת, 30 שנה).
- ניסוח שאלה-תשובה (ה-FAQ בול לזה) → מנועי AI שולפים תשובות ישירות.
- עובדות מובנות (schema) + NAP עקבי → אמון מכונה.

## 7. מדידה
- **Google Search Console** — impressions/clicks/positions למונחי הליבה.
- **GA4** — Key Events: click_call / click_whatsapp / form_submit / click_navigate.
- מעקב דירוג למונחים המקומיים.
