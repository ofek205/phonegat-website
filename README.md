# פון גת — אתר התדמית

אתר סטטי (HTML/CSS/JS) של **פון גת** — מעבדת הסלולר הוותיקה בקרית גת.

## מבנה הפרויקט

| תיקייה / קובץ | תוכן |
|---|---|
| `prototype/` | **האתר עצמו** — זו התיקייה ש-Vercel מגיש לאוויר |
| `prototype/index.html` | הדף הראשי |
| `prototype/privacy.html` | מדיניות פרטיות |
| `prototype/accessibility.html` | הצהרת נגישות |
| `prototype/layers/` | שכבות תמונה לאנימציית ה-Hero |
| `content/` | טקסטים ותוכן (Markdown) — חומר עבודה |
| `.claude/` | הגדרות צוות ה-AI, ידע ומיומנויות — חומר עבודה |

> קבצים כבדים (ארכיוני `.zip`, מסמכי `.pdf`, תמונות מקור) מסוננים ב-`.gitignore` ונשמרים רק במחשב המקומי.

## אירוח ופריסה

האתר מתארח ב-**Vercel** ומחובר למאגר ה-GitHub הזה.
**כל שינוי שנדחף ל-branch `main` מתפרסם אוטומטית לאוויר.**

הגדרת Vercel: **Root Directory = `prototype`**.
